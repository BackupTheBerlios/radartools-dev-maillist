Hi,

I was just scanning through the actual changes. I have some comments:

- rrat.pro: Some common blocks were added. This is not a good idea, because 
rrat should be used also outside of rat on the command line to open rat 
files.

- save_image.pro: Calling channel_defaults (which was added) forces the 
default settings. Therefore it is not written to the pixmap what is seen on 
the screen.

- Tisham: I dont know how you do that, but it is impossible to make a diff 
with files you modified. There is always the entire old file with -tag and 
the new one with +tag. Something in the setting of your editor is odd. We 
should try to find out what. Do you use windows?

Based on the current status I'll prepare the 0.20 release. It will appear in 
the svn soon. Then we should have a quick check if everything is o.k. and 
release the whole stuff. The two points above I would prefer not to include 
for the moment.

Greetings & keep hacking,
Andreas
