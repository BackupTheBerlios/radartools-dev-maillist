Hi All,

I'd like to suggest a freeze of the current state of RAT. There is not much activity anyway, but if somebody can have a check on the sources and finds some bugs, then we could update RAT.

It would be good to bring to conclusion the work of the last two years, which was marked by rather low activity. But still we have new and/or improved imports for F-SAR, ROI_PAC, UAVSAR, ALOS, ENVI; some bugfixes, and batch processing updates.
We could have a release, maybe quiet(?), in 1-2 weeks?

I have a parallel version, which grew a lot, but where I didn't take much care about stability and compatibility. I would like to start to merge it with the official brunch, but this would be best if we finalize the previous work. Some of the issues are geo-coding, rat file format update, visualization, etc. Maybe I'll throw in at some point a "magic forest box" plugin, for the analysis of volumetric data.

Best regards,
Maxim
